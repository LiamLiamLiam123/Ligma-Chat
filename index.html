👑 Admin Commands

/kick <username>
Immediately removes a user from the room.
e.g. /kick John
/mute <username>
Stops a user from sending messages.
e.g. /mute John
/unmute <username>
Lets a muted user talk again.
e.g. /unmute John
/rename <username> <newname>
Changes another user's display name.
e.g. /rename John Dave
/announce <text>
Sends a big highlighted announcement to everyone.
e.g. /announce Welcome everyone!
/pin <text>
Pins a message to the top of the chat. You can also hover any message and click 📌.
e.g. /pin Rules: be nice!
/unpin
Removes the currently pinned message.
e.g. /unpin
/slowmode <seconds>
Sets a cooldown for everyone. Use 0 to disable.
e.g. /slowmode 5
/userslow <username> <seconds>
Sets a slowmode for one specific user only. Use 0 to remove.
e.g. /userslow John 10
/purge <username> <count>
Deletes the last N messages from a user.
e.g. /purge John 5
/warn <username> <message>
Sends a private red warning message only that user can see.
e.g. /warn John Stop spamming
/lock
Stops everyone except admin from sending messages.
e.g. /lock
/unlock
Re-opens the chat after a lock.
e.g. /unlock
/color <username> <color>
Changes a user's name color. Use any CSS color or "reset".
e.g. /color John #ff4757  or  /color John reset
/topic <text>
Sets a topic shown in a bar below the title for everyone.
e.g. /topic Welcome to the room!
/clear confirm
Wipes every message in the room. Requires typing "confirm" to prevent accidents.
e.g. /clear confirm
