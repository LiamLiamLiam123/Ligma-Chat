👑 Admin Commands

/kick <username>
Immediately removes a user from the room.
e.g. /kick John
/mute <username>
Stops a user from sending messages.
e.g. /mute John
/unmute <username>
Lets a muted user talk again.
e.g. /unmute John
/rename <username> <newname>
Changes another user's display name.
e.g. /rename John Dave
/announce <text>
Sends a big highlighted announcement to everyone.
e.g. /announce Welcome everyone!
/pin <text>
Pins a message to the top of the chat.
e.g. /pin Rules: be nice!
