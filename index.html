👑 Admin Commands

/kick <user>
Remove a user from the room.
e.g. /kick John
/mute <user>
Stop a user from sending messages.
e.g. /mute John
/unmute <user>
Let a muted user talk again.
e.g. /unmute John
/warn <user> <message>
Send a private warning only that user sees.
e.g. /warn John Stop spamming
/rename <user> <newname>
Change a user's display name.
e.g. /rename John Dave
/color <user> <color>
Change a user's name color. Use "reset" to remove.
e.g. /color John #ff4757
/announce <text>
Send a highlighted announcement to everyone.
e.g. /announce Welcome!
/pin <text>
Pin a message at the top.
e.g. /pin Rules: be nice
/unpin
Remove the pinned message.
/topic <text>
Set a room topic shown to everyone.
e.g. /topic Friday night chat
/lock
Stop everyone except admin from sending.
/unlock
Re-open the chat after a lock.
/slowmode <seconds>
Set a cooldown for everyone. 0 = off.
e.g. /slowmode 5
/userslow <user> <seconds>
Slowmode for one user only. 0 = off.
e.g. /userslow John 10
/purge <user> <count>
Delete last N messages from a user.
e.g. /purge John 5
/clear confirm
Wipe every message in the room.
