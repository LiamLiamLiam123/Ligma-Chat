­¤ææ Admin Commands

/kick <username>
Immediately removes a user from the room.
e.g. /kick John
/mute <username>
Stops a user from sending messages.
e.g. /mute John
/unmute <username>
Lets a muted user talk again.
e.g. /unmute John
/rename <username> <newname>
Changes another user's display name.
e.g. /rename John Dave
/announce <text>
Sends a big highlighted announcement to everyone.
e.g. /announce Welcome everyone!
/pin <text>
Pins a message to the top of the chat.
e.g. /pin Rules: be nice!
/slowmode <seconds>
Limits how often users can send messages. Use 0 to disable.
e.g. /slowmode 10  |  /slowmode 0
/broadcast <text>
Sends a highlighted message visible in ALL rooms.
e.g. /broadcast Server restart in 5 mins!
